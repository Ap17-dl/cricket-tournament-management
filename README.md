# Cricket Tournament Management Platform

A modern cricket tournament management platform built for local tournaments, college leagues, corporate competitions, and community cricket events.

The platform enables organizers to create tournaments, manage teams, schedule matches, update live scores in real time, and automatically generate player statistics and points tables.

Inspired by the challenges of managing hostel and local cricket tournaments, the goal is to make tournament operations as simple as possible while providing a professional viewing experience.

---

## Core USP

The biggest focus of the platform is the scoring experience.

Instead of navigating through multiple forms and menus, organizers can update matches using a dedicated scoring interface.
